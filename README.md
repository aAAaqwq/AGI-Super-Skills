## 🚀 快速开始

### 安装技能

```bash
# 克隆仓库
git clone https://github.com/aAAaqwq/cc-skills.git ~/.claude/skills

# 或复制到 OpenClaw 工作区
cp -r cc-skills/* ~/clawd/skills/
```

### 使用技能

在 Claude Code 或 OpenClaw 中，技能会自动加载。根据任务描述，AI 会自动选择合适的技能。
