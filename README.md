# Claude Code 全局技能使用指南

本文档列出了所有已安装的全局技能及其使用方法。

## 已安装的技能列表

### 1. n8n 工作流自动化 (n8n-workflow-automation)
**功能**: 自动化创建和管理 n8n 工作流
**使用场景**:
- "创建一个 n8n 工作流来监控邮件并保存附件"
- "优化我的 n8n 工作流"
- "设计一个自动化流程来处理客户数据"

### 2. 网站爬取与 API 自动化 (web-scraping-automation)
**功能**: 自动化爬取网站数据和 API 接口
**使用场景**:
- "爬取这个网站的产品信息"
- "帮我调用这个 API 并解析返回数据"
- "创建一个脚本定时抓取新闻"

### 3. 飞书自动化操作 (feishu-automation)
**功能**: 自动化操作飞书平台(多维表格、消息、文档等)
**使用场景**:
- "在飞书多维表格中添加一条记录"
- "创建一个飞书群组并邀请成员"
- "发送消息到飞书群"

### 4. GitHub 自动化操作 (github-automation)
**功能**: 自动化 GitHub 仓库管理、PR、Issue 等操作
**使用场景**:
- "创建一个新的 GitHub 仓库"
- "提交代码并创建 Pull Request"
- "批量处理 GitHub Issues"

### 5. Figma UI 设计自动化 (figma-ui-design)
**功能**: 自动化 Figma 设计工作流
**使用场景**:
- "创建一个登录页面的 Figma 设计"
- "生成设计系统组件库"
- "将 Figma 设计转换为 React 组件"

### 6. 前端页面开发 (frontend-development)
**功能**: 前端 Web 应用开发
**使用场景**:
- "创建一个 React 登录页面"
- "实现一个响应式导航栏"
- "优化页面加载性能"

### 7. 后端服务开发 (backend-development)
**功能**: 后端 API 和服务开发
**使用场景**:
- "设计一个用户认证 API"
- "创建 RESTful API 接口"
- "设计数据库表结构"

### 8. Chrome 浏览器自动化 (chrome-automation)
**功能**: Chrome 浏览器自动化操作和测试
**使用场景**:
- "自动化登录网站并抓取数据"
- "批量截取网页截图"
- "自动化测试购物流程"

### 9. SEO 文章撰写 (seo-content-writing)
**功能**: 创建 SEO 优化的内容
**使用场景**:
- "写一篇关于 AI 技术的 SEO 文章"
- "优化这篇文章的 SEO 效果"
- "生成产品描述和营销文案"

### 10. UML 图表设计 (uml-diagram-design)
**功能**: UML 图表设计和绘制
**使用场景**:
- "设计一个电商系统的类图"
- "绘制用户登录的时序图"
- "创建系统架构图"

### 11. 项目管理与产品需求 (project-management)
**功能**: 项目管理和产品需求分析
**使用场景**:
- "编写一个电商系统的 PRD"
- "制定项目开发计划"
- "分解功能模块和任务"

## 如何使用技能

### 方法 1: 直接描述需求
Claude 会自动识别并使用合适的技能:
```
我需要创建一个 n8n 工作流来自动化邮件处理
```

### 方法 2: 明确指定技能
如果需要特定技能,可以在描述中提及:
```
使用 GitHub 自动化技能帮我创建一个新仓库
```

### 方法 3: 组合使用多个技能
可以在一个任务中使用多个技能:
```
帮我设计一个电商系统:
1. 先画 UML 类图
2. 编写 PRD 文档
3. 创建前端页面
4. 开发后端 API
```

## 技能目录位置

所有技能存储在: `~/.claude/skills/`

每个技能包含:
- `SKILL.md`: 技能定义和使用说明
- 其他辅助文件(可选)

## 查看技能详情

要查看某个技能的详细说明,可以:
```bash
cat ~/.claude/skills/[技能名称]/SKILL.md
```

例如:
```bash
cat ~/.claude/skills/n8n-workflow-automation/SKILL.md
```

## 技能更新

如需更新技能,直接编辑对应的 `SKILL.md` 文件即可。

## 注意事项

1. **技能自动激活**: Claude 会根据你的需求自动选择合适的技能
2. **技能组合**: 可以同时使用多个技能完成复杂任务
3. **技能优先级**: 如果多个技能都适用,Claude 会选择最匹配的
4. **自定义技能**: 可以参考现有技能创建自己的技能

## 技能开发指南

如需创建新技能,参考以下结构:

```markdown
---
name: skill-name
description: 技能描述和使用场景
allowed-tools: 允许使用的工具列表
---

# 技能标题

## 功能说明
详细描述技能的功能

## 使用场景
列出典型的使用场景

## 技术栈/工具
相关的技术和工具

## 代码示例
提供实用的代码示例

## 最佳实践
使用建议和注意事项
```
